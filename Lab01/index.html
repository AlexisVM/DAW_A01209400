Q&A

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una red de redes, en otras palabras es toda la infraestructura sobre la cual es montada la World Wide Web la cual es un conjunto de información.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET


El método GET requiere una representacion de algun recurso,solo se debe usar para obtener datos.

HEAD


El método HEAD es muy similar al GET solo que en este no se obtiene el body

POST


El método POST nos sirve para mandar datos al servidor.

PUT


Este método es muy similar al un POST pero este crea o remplaza la representación de algun recurso, ademas de ser idempotente.

PATCH


El método PATCH es usado para realizar modificaciones parciales a algun recurso.

DELETE


Este método elimina algun recursos especifico.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?
